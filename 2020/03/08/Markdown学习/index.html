    关于Markdown

    Markdown 是什么

    引用自菜鸟教程

    Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

    Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

    Markdown 编写的文档后缀为 .md, .Markdown。

    Markdown 编辑器

    使用好的工具可以提高效率

    电脑

    Typora

    ​ Typora 支持多平台，格式可直接插入，包含多种主题，编辑后直接渲染出效果，且支持导出多种类型文件。

    Markdown的好处

    • 打开速度快，兼容性问题小，任何文本编辑器都可以打开——Markdown文档其实是纯文本文档，我们看到的美观效果是是最后编辑器渲染的，所以打开迅速，几乎不存在兼容性问题

    • 格式转换方便——使用被称为瑞士军刀的Pandoc可以轻松地将Markdown文档转换为pdf, docx, html等格式

    • 高效——作为一个轻量级的标记语言，Markdown的标记符号较少、简洁，如果配上高效的编辑器，使用快捷键，就更加高效了

    • 培养美感——在阅读完别人的Markdown文档后，你可以学习其格式

    Markdown的缺点

    • 可能在你的编辑器中样式特别漂亮，但是在其他编辑器中就… 解决方法是叫你的好朋友换一个编辑器（狗头）或者是参考别人的格式，编写出普遍觉得好看的
    • 当遇上特殊要求，比如要求一定要是word文档，那就只能转换格式按照要求了，但是……这不是Markdown的缺点OK？
    • 图片是个硬伤——插入图片好像不会嵌入文档，所以分享的时候要把图片一起分享

    Markdown语法
