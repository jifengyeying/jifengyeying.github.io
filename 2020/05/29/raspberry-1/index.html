    常用命令

    sudo raspi-config 配置

    sudo shutdown -h now 关机

    sudo shutdown -r now或者sudo reboot 重启

    cd 文件目录 打开文件目录

    ls 列出目录下的所有文件及文件夹

    cat 文件 读取文件内容

    rm (-rf)+文件夹(目录) 删除文件夹或者目录(-rf)

    mkdir 文件夹 新建文件夹

    cp 文件 目标文件(文件夹) 复制文件到哪里

    cp a.txt /b.txt 把a.txt移动到根目录并重命名为b.txt

    cp a.txt /test 把a.txt移动到test文件夹(前提是有test这个文件，要不会变成根目录的test文件)

    mv 文件 目标文件(文件夹) 移动文件到哪里(同上)
