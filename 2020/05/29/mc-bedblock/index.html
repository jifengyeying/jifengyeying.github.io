    # 服务器教程

    首先你得先购买一个服务器

    可以是你自己的电脑（参见联机教程，学会内网穿透再来），或者是云服务器，这里我选择了阿里云学生机的轻量级服务器，腾讯云等等大同小异，系统选择Ubuntu18.04。

    获得服务器的ip地址后（自己电脑方案的是公网ip，直接跳过ssh）

    ssh - 下载putty和winSCP，方便管理，安装完打开putty，在Host Name里输入ip地址，点击open，提示login as: 输入用户名（阿里云为root），然后输入密码（购买时设置，密码不显示，不用担心）

    mkdir mc - 创建名字为mc的文件夹

    cd mc - 打开文件夹mc

    apt install screen - 安装screen

    apt install aria2 - 安装aria2

    apt install vim - 安装vim

    Java版（插件服）

    首先是服务器核心，这里推荐使用Paper，下载jar，默认最新版，其他版本请点击legacy

    如果你想要使用其他的插件服核心或者mod服核心，你可以在Yive’s Mirror找到其他的。

    点击最新的build，复制下载链接，回到服务器，输入

    下载完成后，制作启动脚本

    start.sh内容

    （如java -Xmx4G -Xms1024M -jar paper-135.jar）

    sh start.sh - 启动服务器

    然后后生成一堆文件，找到 eula.txt ，输入vim eula.txt，将里面的elua=false改为elua=true

    根据需要编辑 server.properties 这个百度

    sh start.sh - 重启服务器

    但是这个时候你只要关闭ssh窗口服务器将关闭，这时就得用screen了

    关闭窗口，成功，享受服务器吧（重新进入screen请输入screen -r 窗口名）

    基岩版（原版生存）

    apt install unzip - 安装unzip

    首先肯定是官方的bds:https://www.minecraft.net/zh-hans/download/server/bedrock

    选择Ubuntu，点击接受用户协议，复制下载链接，存那个文件夹自己建，在服务器输入指令 aria2c 链接 下载

    下载完成后输入 unzip zip文件名

    根据需要编辑 server.properties ，自行百度

    建立一个启动文件 vim start.sh

    输入LD_LIBRARY_PATH=. ./bedrock_server

    然后 screen -S 窗口名（随便） ，输入 sh start.sh 启动服务器，CTRL+a+d ，然后关闭窗口

    进入服务器！（重新进入screen请输入screen -r 窗口名）
